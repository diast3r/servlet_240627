src/main/java : java 클래스
src/main/webapp -> view, jsp, html 등. 앞으로 안 쓸 폴더. Spring 에선 기본적으로 폴더도 없음.
src/main/resources -> 그 외

Spring framework -> boot보다 먼저 나온 것. 설정과정이 엄청나게 복잡함.
Spring boot -> framework의 세팅을 쉽게 만들어 진입장벽을 낮춤. 우리가 쓸 거
사소한 차이 빼고는 사용법이 거의 같아서 구분할 필요 없고,  뭐로 배워도 됨.


처음 서버 실행할 때는 2가지 이유로 에러 발생함.
** 1. debug설정 추가하기
프로젝트 우측 클릭 > Run As > Run Configurations > Arguments 탭 에서
--debug 문구 추가하기.
이 설정을 하려면 적어도 한 번은 서버를 실행하고 난 후여야 함(에러발생하더라도).

**2. db연결 제외
아직 db연동이 되어있지 않은 상태인데, 서버 켤 때 db연결 설정을 자동으로 확인하면서 오류 발생. 
SpringExampleApplication 클래스 위에 어노테이션 (마론달- Db에러 )추가
